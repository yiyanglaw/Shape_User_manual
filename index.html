Shape Classifier App User Manual


1. Introduction
Welcome to the Shape Classifier App! This user manual provides a guide on utilizing the app to classify shapes using a variety of machine learning models. The app facilitates image uploads, shape predictions, and an exploration of model performance through an intuitive interface and data visualization.

2. Getting Started

2.1 System Requirements
The app is designed to run on any system supporting Python. Ensure that Python is installed on your machine.

2.2 Installation(Run in local machine)
- Clone or Download the Repository
- Navigate to the App's Directory
- Install Required Packages: Execute the following command to install the necessary Python packages:
pip install -r requirements.txt

2.3 Running the App
- Run the App: After installing the required packages, run the following command in the terminal or command prompt:
streamlit run app.py
- Open in Browser: The app will open in your default web browser.

3. Shape Classification

3.1 Choosing a Model
- Select Model: Choose a machine learning model from the dropdown list (e.g., SVM, K-NN, NN, Log, Decision_T).

3.2 Uploading Images
- File Uploader: Utilize the file uploader to select multiple images (PNG, JPG, JPEG format). The app displays a reduced-size preview of the uploaded images.

3.3 Initiating Classification
- Classify Button: Click the "Classify" button to predict the shape for the uploaded images. Predictions are displayed for each uploaded image.

4. Dashboard

4.1 Model Performance Overview
- Overview: The dashboard provides an in-depth overview of the selected model's performance. Includes a Classification Report, Confusion Matrix, Precision-Recall Curves, and ROC Curves.

4.2 Model Comparison
- Bar Chart: A bar chart facilitates the comparison of accuracies across different models (e.g., SVM, K-NN, NeuralNetwork, Logistic regression, Decision_Tree).

5. Exploratory Data Analysis (EDA)

5.1 Class Distribution Analysis
- Pie Chart: Explore the distribution of shapes (Circle, Square, Triangle) in the validation dataset through an informative pie chart.
